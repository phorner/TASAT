The DATA folder is where Library Data is stored for each system.
File convention is hexnumber.txt where the file has hard carriage returns at the end of 
each line.  The filename must follow the syntax h####.txt   Example:


filename: h0323.txt
contains:
This is a good system.
The temparature here is great!   What a deal.


The file named:all.txt is for general sector library data.
